## 这页解决什么
教你在“设置”中完成常见配置，提升稳定性和可用性。

## 入口
1. 点击左下角“设置”进入。
2. 页面内含多个配置区域，普通用户优先关注：
   - 模型连接
   - 语言与沉浸式翻译
   - 搜索引擎
   - MCP 服务器

> 截图位置：`S06-01`  
> 建议截图：设置页顶部导航与主要分区。

## 模型连接
1. 填写连接名称、API Key、Base URL、模型名。
2. 点击“测试连接”。
3. 测试通过后点击“保存”。
4. 已有连接可编辑、设为默认或删除。

默认规则说明：
1. 首次新增模型连接后会自动成为默认模型。
2. 后续新增模型连接时，新连接也会自动切换为默认模型。
3. 编辑已有连接不会自动改变默认模型；如需切换，可使用“设为默认”。
4. 如果新连接被自动切换为默认，表单区域会显示“已保存，并切换为默认模型”提示。

> 截图位置：`S06-02`  
> 建议截图：模型连接表单与“测试连接/保存”按钮。

模型命名提示（避免填错）：
1. OpenAI 推荐写法：`GPT-5.4`。
2. Anthropic 推荐写法：`Claude Opus 4.6` 或 `Claude Sonnet 4.6`（不建议写 `Claude 4.7`）。
3. 智谱推荐写法：`GLM-4.7`（或按业务场景使用 GLM-5 系列）。
4. MiniMax 推荐写法：`MiniMax-M2.5`。
5. Moonshot 推荐写法：`Kimi K2` 系列（不建议写 `Kimi K2.5`）。

## 语言与沉浸式翻译
1. 设置默认语言（默认 `zh-CN`）。
2. 按需启用“沉浸式翻译”。
3. 选择翻译显示模式（仅译文或双语内联）。
4. 选择翻译触发方式和翻译引擎策略。
5. 点击“保存语言与翻译设置”。

> 截图位置：`S06-03`  
> 建议截图：语言与翻译配置区域（触发方式与策略）。

## 搜索引擎
1. 切换到“搜索引擎”区域。
2. 通过预设快速选择（如 Brave、Tavily、秘塔等）。
3. 填写 API Key 与必要项后“测试连接”。
4. 点击“保存”。

> 截图位置：`S06-04`  
> 建议截图：搜索引擎预设下拉、测试连接、保存按钮。

## MCP 服务器
1. 切换到“MCP 服务器”区域。
2. 可通过预设快速填写，也可手动填写名称、命令、参数。
3. 按需配置 API Key 或高级环境变量 JSON。
4. 点击“添加 MCP 服务器”。

> 截图位置：`S06-05`  
> 建议截图：MCP 服务器配置表单与“添加 MCP 服务器”按钮。

## 常见问题
1. 配置保存后不生效：关闭设置页后重进确认是否已保存。
2. 翻译失败：优先改为“优先模型，失败回退免费翻译”。
3. 搜索连接失败：确认供应商 API Key 和 Base URL 是否匹配。
4. MCP 添加失败：先检查命令与参数是否可执行。
